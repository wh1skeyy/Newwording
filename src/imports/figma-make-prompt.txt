## SPACED REPETITION LOGIC
Create /src/lib/spaced-repetition.ts

Intervals: [1, 3, 7, 14, 30, 60, 120] days
Labels:    ["Day 1", "Day 3", "Day 7", "Day 14", "Day 30", "Day 60", "Day 120"]

export function generateReviewDates(fromDate: Date): string[] {
  return [1,3,7,14,30,60,120].map(d => {
    const date = new Date(fromDate)
    date.setDate(date.getDate() + d)
    return date.toISOString().split('T')[0]
  })
}

export function getSRStatus(record: SRRecord | null): SRStatus {
  if (!record) return { type: 'not-started', label: 'NOT STARTED' }
  const today = new Date().toISOString().split('T')[0]
  const pending = record.review_dates.filter(
    d => !record.completed_reviews.includes(d)
  )
  if (pending.length === 0) return { type: 'completed', label: 'COMPLETED' }
  const next = pending[0]
  if (next < today) return { type: 'overdue', label: 'OVERDUE' }
  if (next === today) return { type: 'due-today', label: 'DUE TODAY' }
  return { type: 'upcoming', label: `NEXT: ${next}` }
}

## GEMINI AI INTEGRATION
Create /src/lib/gemini.ts

All AI calls go directly to the Gemini 2.5 Flash API.
API key is stored in env var VITE_GEMINI_API_KEY.
Base URL: https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent

const GEMINI_URL = `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=${import.meta.env.VITE_GEMINI_API_KEY}`

async function callGemini(prompt: string): Promise<string> {
  const res = await fetch(GEMINI_URL, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      contents: [{ parts: [{ text: prompt }] }],
      generationConfig: {
        temperature: 0.3,
        maxOutputTokens: 4096,
        responseMimeType: "application/json"
      }
    })
  })
  const data = await res.json()
  return data.candidates[0].content.parts[0].text
}

### FUNCTION 1: enrichWords(words: string[]): Promise<Word[]>

Prompt to send to Gemini:
"""
You are an English vocabulary expert for Vietnamese learners.
Given this list of English words, return a JSON array where each item has:
- "word": the original English word (string)
- "vietnamese": the most common Vietnamese translation (string)
- "type": the grammatical type — use only: "n", "v", "adj", "adv", "phr v", "phrase" (string)
- "example": one natural example sentence using the word at B1/B2 level (string)

Return ONLY a valid JSON array. No explanation, no markdown, no code blocks.

Words: ${words.join(', ')}
"""

Parse the response string as JSON → return as Word[].
On error: return words.map(w => ({ word: w, vietnamese: '...', type: 'n', example: '...' }))

### FUNCTION 2: generateSentences(words: Word[]): Promise<SentenceItem[]>

interface SentenceItem {
  sentence: string    // sentence with "______" as the blank
  answer: string      // the correct word
  options: string[]   // all word.word values from the lesson, shuffled
}

Prompt to send to Gemini:
"""
You are an English teacher creating fill-in-the-blank exercises for Vietnamese B1/B2 students.

Given these vocabulary words (with meanings), create one fill-in-the-blank sentence for EACH word.

Rules:
- Each sentence must be 8-15 words long
- Use "______" (6 underscores) as the blank
- The sentence context must make the correct answer clear
- Write at B1/B2 level — natural, not overly formal
- The blank must be filled by exactly the word provided (not a synonym)
- Return ONLY a valid JSON array, no explanation, no markdown, no code blocks

Each item in the array must have:
- "sentence": the sentence string containing "______"
- "answer": the correct word that fills the blank

Words and meanings:
${words.map(w => `- ${w.word} (${w.type}): ${w.vietnamese} — "${w.example}"`).join('\n')}
"""

Parse response as JSON array of { sentence, answer }.
Then add "options" to each item: all lesson words shuffled (so student picks from full word list).
Return as SentenceItem[].

## SUPABASE HELPER
Create /src/lib/supabase.ts:

import { createClient } from '@supabase/supabase-js'

export const supabase = createClient(
  import.meta.env.VITE_SUPABASE_URL,
  import.meta.env.VITE_SUPABASE_ANON_KEY
)

export async function getNextId(key: 'students' | 'lessons'): Promise<string> {
  const { data } = await supabase.rpc('increment_counter', { counter_key: key })
  return String(data).padStart(4, '0')
}

Also create this Postgres function in Supabase SQL editor:

CREATE OR REPLACE FUNCTION increment_counter(counter_key text)
RETURNS int AS $$
  UPDATE id_counters SET value = value + 1
  WHERE key = counter_key
  RETURNING value;
$$ LANGUAGE sql;

## IMPORTANT NOTES

- ALL Supabase operations are client-side (no backend needed with Figma Make)
- Enable Row Level Security (RLS) in Supabase but set permissive policies
  for now since there is no auth system:
  CREATE POLICY "allow all" ON [table] FOR ALL USING (true);
  Apply this to: classes, students, lessons, master_words, spaced_repetition, id_counters

- 4-digit IDs always zero-padded: String(n).padStart(4, '0')

- b1Words array is a placeholder — will be swapped with ~3000 words later.
  For sentence making B1 pairs, pick 2 DIFFERENT random words per new vocab word:
  const shuffled = [...b1Words].sort(() => Math.random() - 0.5)
  const pair = shuffled.slice(0, 2)

- Gemini API temperature is set to 0.3 for consistent, predictable output.
  Always request responseMimeType: "application/json" to get clean JSON back.
  Wrap all Gemini calls in try/catch. On failure show error toast and allow retry.

- Sentences tab in lesson page: generate on lesson mount, cache to Supabase.
  Sentences tab in practice page: generate fresh each session, no caching.

- Skeleton loaders on ALL async data fetches. Never show empty states
  during loading — always show pulsing skeletons.

- Mobile responsive:
  - Matching exercise: stack columns vertically on mobile
  - Lesson cards: stack vertically
  - Table: horizontal scroll on mobile
  - Teacher forms: full width single column on mobile

- Invisible teacher entry div must have:
  style={{ position:'fixed', top:0, right:0, width:40, height:40,
  zIndex:9999, background:'transparent', border:'none', cursor:'default',
  outline:'none' }}
  tabIndex={-1} — absolutely no visual indicator whatsoever

- Toast notifications:
  Bottom-right corner, bg #272a2c, border #464554, 8px radius,
